# Site-Institucional-FGC
Fundação Gabriel de Campos – Site Institucional

📄 Sobre o Projeto

Este projeto é um site institucional da Fundação Gabriel de Campos, desenvolvido para apresentar informações institucionais, projetos, campanhas e meios de contato. O site é responsivo, utiliza cores vermelho e branco como identidade visual e contém um formulário de contato com validação via JavaScript.

⚙ Funcionalidades

Menu de navegação centralizado e intuitivo.

Logo da fundação em destaque na página inicial.

Formulário de contato com validação:

Alerta para campos obrigatórios em branco.

Mensagem “Formulário enviado com sucesso!” quando preenchido corretamente.

Layout responsivo para desktop e dispositivos móveis.

🛠 Tecnologias

HTML5 – Estrutura semântica das páginas.

CSS3 – Estilização visual, cores, fontes e layout responsivo.

JavaScript – Validação e interatividade do formulário.
